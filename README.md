# Library Management Data Analysis

## Project Overview
This project analyzes library data to uncover trends in borrowing habits and member demographics. The insights generated are designed to help library management optimize their purchasing budget and improve member services.

## Dataset
This analysis uses three datasets:
* `books.csv`: Contains master inventory details (Genre, Author, Publication Year).
* `members.csv`: Contains demographic data of registered library users.
* `borrowings.csv`: The transaction log of checkouts and returns.

## How to View and Run
Because this project was developed in Google Colab, you do not need to install Python or Pandas on your local machine to view it
